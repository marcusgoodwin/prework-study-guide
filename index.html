HTML

  • HTML is the main code used to format a website.
  • It also is what is considered the foundation of said website.
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • CSS is commonly used to format the visual aspect of the webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is being used as a style folder on this current webpage.

GIT

  • Using GIT, you can work on the same code as your partners without interupting what they're doing
  • GIT also allows you to connect GITHub to your PC, and allows you to store your work on the cloud
  • git status: checks what branch we are currently working on
  • git checkout -b branch-name: creates a new branch and switches to it
